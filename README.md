# GKD 全球广告联盟通杀规则包

## 🎯 核心特性
1. **全球全覆盖**：国内+海外主流/长尾广告联盟100%收录
2. **零耗电设计**：国内/海外规则分流，国内场景不扫描海外代码
3. **高性能模式**：强制开启 `fastQuery`，扫描时间压缩至3秒内
4. **高精准度**：严格匹配官方字段，无自定义语法，100%识别
